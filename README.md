# 🎓 Student Management System

> "Education is the passport to the future, for tomorrow belongs to those who prepare for it today."

A simple **Console-Based Student Management System** developed to manage student records efficiently using a command-line interface. This project allows users to perform basic CRUD operations such as adding, viewing, updating, and deleting student information.

---

## 📌 Project Overview

The Student Management System is designed to help manage student data through a console application. It provides an easy way to store and maintain student records while demonstrating fundamental programming concepts.

---

## ✨ Features

- ➕ Add New Student Records
- 📋 View All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- ❌ Delete Student Records
- 🧮 Menu-Driven Console Interface
- 💾 Data Management Using File Handling / Collections
- ⚠️ Input Validation and Error Handling



